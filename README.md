# final-project-AI-sm

აღწერა: გამოვიყენე XLM-RoBERTa მოდელი და გავმართე, რომ შეეძლოს სოციალური მედიის კომენტარის დადებითი, უარყოფითი ან ნეიტრალური განწყობის განსაზღვრა.

როგორ ჩავრთოთ: საჭიროა .ipynb ფაილის Colab-ში გახსნა. შევქმენი Gradio-ს აპლიკაციისთვის გამოსაყენებლად.
